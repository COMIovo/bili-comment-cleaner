# B站评论清理助手

这是一个 Chrome / Edge Manifest V3 扩展，用 AICU 的历史评论索引定位当前登录 B站账号的过往视频评论，再通过 B站登录态进行预览、核验、勾选删除和日志导出。

## 当前能力

- 读取当前浏览器的 B站 `DedeUserID` 与 `bili_jct`，不保存账号密码。
- 从 `api.aicu.cc` 分页拉取当前 UID 的历史评论索引。
- 支持关键词、日期范围、限定视频列表筛选。
- 支持一级评论和二级回复候选项。
- 删除前默认调用 B站评论详情接口核验评论仍属于当前 UID。
- 删除时按间隔逐条执行，遇到未登录、CSRF 错误或请求频繁会停止。
- 支持 CSV / JSON 导出完整候选和处理日志。
- 成功删除、已不存在、归属不符的评论会写入本地已处理记录库，后续扫描默认隐藏，避免重复跳过。
- 可切换“显示已处理评论”，也可清除当前 UID 的本地已处理记录。
- B站接口请求会通过一个 `bilibili.com` 工作标签页执行，尽量避免扩展页跨站请求不携带登录 cookie 的问题。
- AICU 扫描请求会通过一个 `aicu.cc` 工作标签页执行，避免扩展页来源触发 `HTTP 468`。
- AICU 扫描恢复直连为主路径，工作页/API 页签只作为失败兜底。
- 如果 AICU 工作页脚本没有返回结果，会自动改用 `api.aicu.cc` 页签读取 JSON。
- 可复制诊断日志，方便排查删除接口返回码、核验失败和过期索引。
- 对 AICU 返回的对象型 `root` 做兼容处理，并在严格核验时使用 `seek_rpid` 兜底。

## 安装方法

1. 打开 Edge 或 Chrome 的扩展程序页面。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录 `bili-comment-cleaner`。
5. 登录 B站后点击扩展图标，进入独立管理页。

## 使用建议

1. 先刷新登录态，确认显示当前 UID。
2. 设置关键词、时间范围或限定视频列表。
3. 点击“扫描评论”，检查预览表。
4. 只勾选确定要删除的评论。
5. 保持默认严格核验和较慢删除间隔。
6. 删除结束后导出 CSV 或 JSON 留档。

## 边界说明

- 第一版只支持普通视频评论，评论区类型固定为 `type=1`。
- AICU 数据不是实时数据，可能存在漏评论或已删除评论仍在索引里的情况。
- 已处理记录仅保存在本机浏览器的扩展存储中，更换浏览器或清空扩展数据后需要重新积累。
- 严格核验失败的评论会被跳过，不会继续删除。
- B站接口和 AICU 接口都可能变动，如出现大面积失败，需要更新适配逻辑。
